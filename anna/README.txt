save_srs_to_jira_fields parses a csv file with srs ids to jira tickets
It then saves all the srs ids to those jira tickets

To run save_srs_to_jira_fields.py type ./save_srs_to_jira_fields.py into the command line
The file that's used for mappings is currently hardcoded into the code

Before running the file, change auth.cfg to your atlassian username and password that you use to access the corresponding jira board.
